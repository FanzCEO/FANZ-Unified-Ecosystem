API & Integration Notes (high level)
	•	REST/GraphQL for internal clients; Webhooks for payment gateways, SMS, email, card issuer, NFT chain.
	•	Event Bus (e.g., Kafka/NATS) between uploads → transcode → publish; purchase → unlock → notify; tip → alert → analytics.
	•	Search via OpenSearch/Meilisearch for creators, clips, events, threads.
	•	Real-time via WebSockets for chat, live tips, contests, live battles.
	•	Storage: object storage (S3-compatible) + signed URLs; CDN with tokenized paths.

⸻

Data/Compliance (quick checklist)
	•	Age/KYC (creator + payout), geo-blocking, content classification, DMCA/2257 vault, export/download request handler (GDPR/CCPA), audit logs, payment PCI separation.
