Proprietary Feature — Geolocation Map with Creator‑Only Chat

The FansLab Map UX
	•	Private Creator Layer: A live, interactive map displaying verified creators within adjustable radius (privacy settings configured by creators).
	•	Functionalities:
	•	Real-Time Presence Indicators (online/offline).
	•	“Open for Collab/Meet” Pins: Creators broadcast availability for meetups or co-creation.
	•	Direct Chat From Map: Tap a pin to open instant creator-to-creator chat, propose collabs, or schedule visits.
	•	Filterable by Niche, Ratings, Events: Use RateLab status, upcoming event pins, or topic filters.
	•	Integration Touchpoints:
	•	ForumLab: Open discussions or invite peers directly via map.
	•	ViserGo: Launch location-based contests (e.g. “Best local shoot”).
	•	Apointer/EventLab: Schedule nearby workshops or private sessions via map interaction.
