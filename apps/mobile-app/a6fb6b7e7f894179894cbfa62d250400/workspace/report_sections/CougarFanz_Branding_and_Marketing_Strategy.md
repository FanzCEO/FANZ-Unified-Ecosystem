#### Logo Integration Guidelines

When integrating the provided logo into the CougarFanz brand system:

**Logo Placement**:
- Primary position: Top-left of navigation header with 40-60px height on desktop, 32-40px on mobile
- Alternative position: Centered on landing pages and marketing materials
- Minimum clear space: Equal to the height of the logo mark on all sides
- Minimum size: 24px height to maintain legibility

**Logo Variations**:
- Full-color version on light backgrounds (primary use)
- White/cream version on dark backgrounds (navigation, footers)
- Black version for print materials and single-color applications
- Monochrome version maintaining brand recognition in limited-color contexts

**Logo Don'ts**:
- Never rotate, skew, or distort the logo proportions
- Never apply drop shadows, gradients, or effects
- Never place on busy backgrounds that reduce legibility
- Never recreate or modify logo elements
- Maintain minimum size requirements across all applications

#### Typography System

**Primary Typeface (Headings & Display)**:
Select a sophisticated serif font (e.g., Playfair Display, Cormorant, Lora) or modern sans-serif with elegant proportions (e.g., Montserrat, Raleway, Poppins) that conveys confidence and premium quality. Serif fonts add traditional elegance and authority, while modern sans-serifs provide clean sophistication.

**Secondary Typeface (Body & UI)**:
Choose a highly legible sans-serif font (e.g., Inter, Open Sans, Source Sans Pro) for body text, interface elements, and extended reading. Prioritize readability across devices and screen sizes.

**Typography Hierarchy**:

| Element | Size | Weight | Line Height | Usage |
|---------|------|--------|-------------|-------|
| H1 Display | 48-64px | Bold (700) | 1.2 | Landing pages, hero sections |
| H2 Section | 36-42px | Semibold (600) | 1.3 | Major section headers |
| H3 Subsection | 28-32px | Semibold (600) | 1.4 | Subsection headers, cards |
| H4 Component | 20-24px | Medium (500) | 1.4 | Component headers, labels |
| Body Large | 18-20px | Regular (400) | 1.6 | Feature descriptions, introductions |
| Body Standard | 16px | Regular (400) | 1.6 | Primary body text |
| Body Small | 14px | Regular (400) | 1.5 | Secondary text, captions |
| UI Elements | 14-16px | Medium (500) | 1.4 | Buttons, navigation, forms |

**Typography Guidelines**:
- Maintain consistent hierarchy across all platform touchpoints
- Use sentence case for most UI elements; title case for major headings
- Limit font weights to 3-4 variations to maintain visual consistency
- Ensure minimum 4.5:1 contrast ratio for body text (WCAG AA compliance)
- Optimize line length to 50-75 characters for optimal readability
