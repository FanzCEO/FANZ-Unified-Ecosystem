## ğŸ†˜ Troubleshooting

### Upload Failed
**Solutions:**
1. Check internet connection
2. Verify file size (max 2GB for videos)
3. Ensure file format is supported (JPG, PNG, GIF, MP4, MOV)
4. Try compressing large files
5. Contact support if issue persists

### File Processing Slowly
**Why?**
- Large file size (videos take longer)
- Server is processing watermark/compression
- Typically completes within 1-2 minutes

**What to do:**
- Upload continues in background
- You'll get a notification when ready
- Check "Upload Queue" in profile menu

### Content Rejected by AI Moderation
**Possible Reasons:**
- Violates community guidelines
- Missing age verification
- Flagged for suspicious content
- Technical issue

**Next Steps:**
1. Review community guidelines
2. Ensure all performers are age-verified
3. Appeal decision if you believe it's an error
4. Contact support for clarification
