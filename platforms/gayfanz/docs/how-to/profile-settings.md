### "Profile photo won't upload"

**Common issues:**
- File too large (max 5MB)
- Wrong format (use JPG/PNG)
- Poor internet connection
- Browser cache issue

**Solutions:**
1. Compress image (under 5MB)
2. Convert to JPG
3. Clear browser cache
4. Try different browser
5. Contact support
