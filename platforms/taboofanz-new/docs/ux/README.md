# TabooFanz UX Specifications

## Design Philosophy

TabooFanz embodies a **"neon noir + cyber fetish + goth club"** aesthetic. The platform feels edgy and forbidden while remaining professional and safe-by-design.

### Core Principles

1. **Dark-First**: Ultra-dark backgrounds with neon accents
2. **Consent-Visible**: Boundaries and guidelines always visible
3. **Identity-Protective**: Masked mode and privacy are first-class features
4. **Archetype-Driven**: Personas influence visual treatment
