# TransFanz

**The gender-diverse heart of the FANZ network**

TransFanz is an 18+ creator platform dedicated to trans, nonbinary, gender-expansive, and questioning creators and their fanz.

## Core Values

- **Gender-Affirming**: Pronouns and chosen names are central to every experience
- **Safe**: Strong anti-harassment protections with trans-specific safety features
- **Empowering**: Creator-first design that celebrates authentic self-expression
- **Inclusive**: All gender identities welcome - trans women, trans men, nonbinary, genderfluid, agender, and more

## Platform Features

### For Creators
- Chosen name displayed everywhere (legal name only for compliance)
- Pronoun pills prominently displayed
- Persona archetypes (The Phoenix, The Enby Star, The Shapeshifter, etc.)
- Control over pre-transition content visibility
- AI-powered affirming bio and caption suggestions
- Dedicated safety dashboard with harassment filters

### For Fans
- Discover diverse trans creators respectfully
- Filter by identity, expression, vibe, and content style
- Transparent recommendations ("Why you're seeing this")
- Support creators directly through subscriptions and tips

### Safety First
- Transphobia as priority moderation category
- Auto-detection of transphobic slurs
- One-tap block & report
- Region blocking for creator protection
- 4-hour SLA for urgent safety reports

## Tech Stack

- **Frontend**: Next.js 14+ (App Router), React, Tailwind CSS
- **API**: tRPC + REST
- **Database**: PostgreSQL with Prisma ORM
- **Cache**: Redis
- **Storage**: S3-compatible with CDN
- **Real-time**: Socket.io for live features
- **AI**: OpenAI + custom safety models

## Design System

TransFanz uses a trans-pride inspired but matured color palette:

- `tfz-bg`: Deep midnight purple
- `tfz-accent-primary`: Trans-pink inspired soft accent
- `tfz-accent-secondary`: Trans-blue calming accent
- `tfz-accent-tertiary`: Lavender/magenta highlights

Visual motifs include soft gradients, aurora/nebula backgrounds, and sparkle effects representing transformation and possibility.
